# WeatherApp

This is a weather app written by Ailwyn Liang. 

## Environment:

minSdkVersion      21  
targetSdkVersion   30   

## Development tools:

Android Studio

## Describe:

The basic techs of this app are MVVM, Retrofit, ViewModel, DataBinding, LiveData and Room.  
The coding language of this app is Kotlin.
